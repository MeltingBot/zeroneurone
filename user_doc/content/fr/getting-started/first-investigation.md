# Créer votre première enquête

Une **enquête** dans ZeroNeurone est un espace de travail complet contenant vos éléments, liens, vues et fichiers joints.

## Créer une enquête

1. Depuis la page d'accueil, cliquez sur **Nouvelle enquête**
2. Donnez un nom à votre enquête
3. (Optionnel) Ajoutez une description


## Ouvrir une enquête existante

Cliquez sur une carte d'enquête dans la liste pour l'ouvrir.


## Organiser vos enquêtes

### Recherche et filtres

La barre de recherche permet de filtrer les enquêtes par nom ou description. Combinez avec des filtres par tags pour retrouver rapidement vos enquêtes.

### Tags d'enquête

Associez des tags à vos enquêtes pour les organiser par thème, client, statut, etc.

1. Cliquez sur le menu **⋯** d'une carte
2. Sélectionnez **Gérer les tags**
3. Ajoutez des tags existants ou créez-en de nouveaux
4. Les tags ajoutés servent de filtres sur la page d'accueil

Exemples de tags utiles :
- Par statut : `En cours`, `Archivé`, `Urgent`
- Par thème : `Cyber`, `Fraude`, `RH`
- Par client : `Client-A`, `Projet-X`

### Favoris

Cliquez sur l'étoile d'une carte pour marquer une enquête comme favorite. Les favoris apparaissent en premier dans la liste.

### Tri

Utilisez le sélecteur de tri pour ordonner vos enquêtes :
- **Récent** — Par date de modification (défaut)
- **Créé** — Par date de création
- **Nom** — Alphabétique


## Actions sur une enquête

Cliquez sur le menu **⋯** d'une carte pour :

- **Renommer** — Modifier le nom et la description
- **Gérer les tags** — Ajouter ou supprimer des tags
- **Exporter ZIP** — Sauvegarder en archive ZIP
- **Supprimer** — Supprimer définitivement (avec confirmation)

{{< hint warning >}}
**Attention** : La suppression est irréversible. Pensez à exporter vos données avant.
{{< /hint >}}

## Structure d'une enquête

Une enquête contient :

| Composant | Description |
|-----------|-------------|
| **Éléments** | Les nœuds de votre graphe (personnes, lieux, objets...) |
| **Liens** | Les relations entre éléments |
| **Vues** | Configurations de filtres et d'affichage sauvegardées |
| **Fichiers** | Documents, images, PDF joints aux éléments |
