# Créer votre première enquête

Une **enquête** dans ZeroNeurone est un espace de travail complet contenant vos éléments, liens, vues et fichiers joints.

## Créer une enquête

1. Depuis la page d'accueil, cliquez sur **Nouvelle enquête**
2. Donnez un nom à votre enquête
3. (Optionnel) Ajoutez une description


## Ouvrir une enquête existante

Cliquez sur une carte d'enquête dans la liste pour l'ouvrir.


## Actions sur une enquête

Cliquez sur le menu **⋯** d'une carte pour :

- **Renommer** — Modifier le nom et la description
- **Dupliquer** — Créer une copie complète
- **Exporter** — Sauvegarder en ZIP, JSON ou autre format
- **Supprimer** — Supprimer définitivement (avec confirmation)

{{< hint warning >}}
**Attention** : La suppression est irréversible. Pensez à exporter vos données avant.
{{< /hint >}}

## Structure d'une enquête

Une enquête contient :

| Composant | Description |
|-----------|-------------|
| **Éléments** | Les nœuds de votre graphe (personnes, lieux, objets...) |
| **Liens** | Les relations entre éléments |
| **Vues** | Configurations de filtres et d'affichage sauvegardées |
| **Fichiers** | Documents, images, PDF joints aux éléments |
