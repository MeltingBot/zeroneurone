# Raccourcis clavier

Liste complète des raccourcis clavier de ZeroNeurone.

## Navigation

| Raccourci | Action |
|-----------|--------|
| **Ctrl+K** | Recherche rapide |
| **1** | Vue Canvas |
| **2** | Vue Carte |
| **3** | Vue Timeline |
| **4** | Vue Matrice |
| **Échap** | Fermer modal / Désélectionner |

## Sélection

| Raccourci | Action |
|-----------|--------|
| **Clic** | Sélectionner un élément |
| **Ctrl+clic** | Ajouter/retirer de la sélection |
| **Ctrl+A** | Sélectionner tout |
| **Rectangle** | Sélection par zone (clic-glisser sur le fond) |

## Édition

| Raccourci | Action |
|-----------|--------|
| **Suppr** / **Retour** | Supprimer la sélection |
| **Ctrl+Z** | Annuler |
| **Ctrl+Shift+Z** | Rétablir |
| **Ctrl+Y** | Rétablir (alternatif) |
| **Ctrl+C** | Copier |
| **Ctrl+V** | Coller |
| **Ctrl+D** | Dupliquer |

## Canvas

| Raccourci | Action |
|-----------|--------|
| **Double-clic** | Créer un élément |
| **Molette** | Zoom |
| **Ctrl+molette** | Zoom fin |
| **Glisser fond** | Déplacer la vue |
| **F** | Mode Focus (voisinage) |
| **Ctrl+0** | Zoom pour tout voir |
| **Ctrl++** | Zoom avant |
| **Ctrl+-** | Zoom arrière |

## Vue Matrice

| Raccourci | Action |
|-----------|--------|
| **Double-clic** | Editer une cellule |
| **Entree** | Confirmer l'edition |
| **Echap** | Annuler l'edition |
| **Tab** / **Shift+Tab** | Colonne suivante / precedente |
| **Fleche Haut/Bas** | Ligne adjacente (en edition) |
| **Ctrl+C** | Copier les lignes selectionnees en TSV |
| **Ctrl+clic** | Ajouter/retirer de la selection |
| **Shift+clic** | Selection par plage |

## Panneaux

| Raccourci | Action |
|-----------|--------|
| **[** | Basculer panneau gauche |
| **]** | Basculer panneau droit |

## Fichiers

| Raccourci | Action |
|-----------|--------|
| **Ctrl+S** | Exporter (ZIP) |
| **Ctrl+O** | Importer |

---

## Personnalisation

Les raccourcis ne sont pas personnalisables dans la version actuelle.
