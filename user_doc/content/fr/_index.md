# ZeroNeurone

**Tableau d'analyse visuelle pour enquêteurs et analystes**

{{< hint >}}
Version actuelle : **1.5**
{{< /hint >}}

ZeroNeurone est un outil d'amplification cognitive qui combine la simplicité d'un tableau blanc infini avec la puissance de l'analyse de graphes. Conçu pour les analystes, enquêteurs et chercheurs qui ont besoin de visualiser et explorer des relations complexes.


## Principes fondamentaux

- **L'humain reste maître** — Aucune action automatique, suggestions uniquement sur demande
- **100% local** — Vos données restent sur votre machine, fonctionne hors-ligne
- **Le visuel EST l'analyse** — Position, couleurs, formes portent du sens
- **Zéro ontologie imposée** — Créez vos propres concepts, aucun type d'entité forcé

## Fonctionnalités principales

| Fonctionnalité | Description |
|----------------|-------------|
| **Canvas infini** | Tableau blanc zoomable et pannable |
| **Liens riches** | Relations avec métadonnées, dates, confiance |
| **Vue carte** | Positionnement géographique des éléments |
| **Timeline** | Visualisation chronologique |
| **Analyse de graphe** | Clusters, centralité, chemins |
| **Collaboration** | Temps réel, chiffré de bout en bout |
| **Import/Export** | ZIP, CSV, JSON, GraphML, GeoJSON |
