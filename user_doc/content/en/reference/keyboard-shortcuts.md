# Keyboard Shortcuts

Complete list of ZeroNeurone keyboard shortcuts.

## Navigation

| Shortcut | Action |
|----------|--------|
| **Ctrl+K** | Quick search |
| **1** | Canvas view |
| **2** | Map view |
| **3** | Timeline view |
| **Escape** | Close modal / Deselect |

## Selection

| Shortcut | Action |
|----------|--------|
| **Click** | Select an element |
| **Ctrl+click** | Add/remove from selection |
| **Ctrl+A** | Select all |
| **Rectangle** | Area selection (click-drag on background) |

## Editing

| Shortcut | Action |
|----------|--------|
| **Delete** / **Backspace** | Delete selection |
| **Ctrl+Z** | Undo |
| **Ctrl+Shift+Z** | Redo |
| **Ctrl+Y** | Redo (alternative) |
| **Ctrl+C** | Copy |
| **Ctrl+V** | Paste |
| **Ctrl+D** | Duplicate |

## Canvas

| Shortcut | Action |
|----------|--------|
| **Double-click** | Create an element |
| **Wheel** | Zoom |
| **Ctrl+wheel** | Fine zoom |
| **Drag background** | Pan view |
| **F** | Focus mode (neighborhood) |
| **Ctrl+0** | Zoom to fit all |
| **Ctrl++** | Zoom in |
| **Ctrl+-** | Zoom out |

## Panels

| Shortcut | Action |
|----------|--------|
| **[** | Toggle left panel |
| **]** | Toggle right panel |

## Files

| Shortcut | Action |
|----------|--------|
| **Ctrl+S** | Export (ZIP) |
| **Ctrl+O** | Import |

---

## Customization

Shortcuts are not customizable in the current version.

{{< hint info >}}
**Tip**: Hover over buttons to see their keyboard shortcut.
