# ZeroNeurone

**Visual analysis board for investigators and analysts**

{{< hint >}}
Current version: **1.4**
{{< /hint >}}

ZeroNeurone is a cognitive amplification tool that combines the simplicity of an infinite whiteboard with the power of graph analysis. Designed for analysts, investigators, and researchers who need to visualize and explore complex relationships.


## Core Principles

- **Human stays in control** — No automatic actions, suggestions only on demand
- **100% local** — Your data stays on your machine, works offline
- **Visual IS the analysis** — Position, colors, shapes carry meaning
- **Zero imposed ontology** — Create your own concepts, no forced entity types

## Main Features

| Feature | Description |
|---------|-------------|
| **Infinite canvas** | Zoomable and pannable whiteboard |
| **Rich links** | Relationships with metadata, dates, confidence |
| **Map view** | Geographic positioning of elements |
| **Timeline** | Chronological visualization |
| **Graph analysis** | Clusters, centrality, paths |
| **Collaboration** | Real-time, end-to-end encrypted |
| **Import/Export** | ZIP, CSV, JSON, GraphML, GeoJSON |
